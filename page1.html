Элементы и атрибуты XML

XML (eXtensible Markup Language) — это язык разметки, который используется для хранения и передачи структурированных данных.
В отличие от HTML, XML не задаёт визуальное оформление — его задача заключается в описании структуры данных. Основные строительные блоки XML — это элементы и атрибуты.
